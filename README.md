# ChatZone
A realtime chat app created using nodejs, express, socketio and vanilla js. I followed this tutorial https://www.youtube.com/watch?v=jD7FnbI76Hg

![screenshot]

## Usage
* git clone https://github.com/mecitdeniz/chatZone.git
* npm install
* npm dev
